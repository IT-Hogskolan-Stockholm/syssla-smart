# syssla-smart
En todo-app för hushåll med flera personer som ofta bråkar om, eller glömmer bort, vem som ska göra vardagssysslor som städning, matlagning eller handla.

## Project Setup

 ```sh
npm install
```

### JSON-Server

```sh
npm run server
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
